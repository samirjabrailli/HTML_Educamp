Front End haqda qısa məlumat

Frontend və backend inkişafları bir-birindən fərqlənsə də onlar sanki eyni
bir qəpiyin iki tərfidir. Veb saytın funksionallığı bir tərəfdə ünsiyyət quraraq
digər tərəfdən effektiv fəaliyyət göstərən tərəflərə əsaslanır. Onların veb proqramlaşdırmada rolları böyükdür. Bəs nədən başlamalıyıq ?

Front End nədir

Front end veb brouzerlərdə görünən hissədir.
Həmçinin “client-side” (istifadəçi hissə) də adlandırılır.
O özündə hər şeyi : yazıları , rəngləri , düymələri , şəkilləri və nəzarətçi meyunu da ; birləşdirir.

Front Endə nələr aiddir ?

  • HTML - Frontend üçün fundemental dildir. Veb brouzerlər bu dil sayəsində işləyirlər. HTML brouzerlərdə displaydə əks olunur
  • CSS - HTML dilini müşayiət edir . CSS-lə sən veb saytlarda stilləri dəyişə bilərsən (rəngləndirmə , responsivləşdirmə və s.) hansı ki rəngləri , fontları dəyişməkdə kömək edir.
  • BOOTSTRAP - Websaytda responsivlik üçün istifadə edilir
  • Javascript - menyular, modal pəncərələr və kontakt formaları kimi daha interaktiv elementlər üçün istifadə olunan proqramlaşdırma dilidir.
  • REACT JS və NEXT JS - daha profosional saytlar yaratmaq üçün istifadə eilir.
